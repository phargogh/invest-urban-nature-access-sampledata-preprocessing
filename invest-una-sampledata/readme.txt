
"population.tif" is the number of population in a pixel. I excluded pixels without people to reduce compulation load. 

"Commune.shp" defines the spatial unites that aggreated level of supply-demand indicators should be calculated. Also this layer defines the population structure which will be used in "split population" module. Here the population is split into different gender and age(old and young).

"Greenspace.tif" is the greenspace layer. The land use type related to the land use code can be found in Paris manuscript submitted to Land Use Policy. 

Details of data sources and process can be found in Paris manuscript submitted to Land Use Policy.
